## 这是什么

AI 写的东西，往往一眼就能看出来：排比堆叠、空心大词、面面俱到、工整收尾，
读着正确，但塑料感很重；更糟的是它**谁看都一样**——给业务 leader 和给研发 leader
是同一篇，而且内行一看就觉得肤浅。

Bragi 一台引擎，两种用法：**改写**你的 AI 草稿，或**从你的真实素材 0→1** 产出一份
汇报/方案/文档（给素材 + "产出什么、给谁看"，它从料里抽真材料搭出来，素材没有的会问你）。
两种都做下面三件事：

1. **去 AI 味** —— 按一张"破绽清单"铲掉排比、空心大词、万能过渡句这些一眼能认出 AI 的痕迹。
2. **对准角色** —— 动笔前先问"这稿子给谁看"，明确后按那个人在乎的东西重排重点。
3. **逼出垂类纵深** —— 把外行也能写的泛泛之谈标出来，向你要真东西（具体数字、真实机制、
   内行才知道的取舍），**给了就织进去，给不了就如实标"缺料"，绝不凭空编造**。

> 名字取自北欧神话的诗歌与雄辩之神、众神的吟游诗人 Bragi：同一段史诗，他在国王的厅堂
> 和在战士的火堆旁讲法不同。和 [Muninn](https://github.com/uaandesign/muninn-skill-leveler) 同源——
> Muninn 管记忆与审计，Bragi 管措辞与表达。

## 它怎么改（三层）

**先问，再改。** 不知道"给谁看"就不动笔——同一份内容，给业务 leader 和给研发 leader
是两篇文章。问清楚之后：

| 层 | 干什么 | 关键 |
|---|---|---|
| **去 AI 味** | 铲两层：土味 slop（赋能/闭环/排比）+ **聪明改写腔**（对仗金句、生造比喻、"一句话："开场） | 减法，靠删和断；对照 `voice.md` 往真人写法上靠，别用对仗去"显得像人" |
| **对准角色** | 换他听得进的讲法把整件事**讲完并卖出去**，价值翻译成他在乎的结果 | **不是只留他的领域删掉其余**——是什么/怎么做/验证/进度/下一步，对谁都保留 |
| **逼出纵深** | 标出泛化句 → 向你要真料 → 织入或标缺料 | **不造料，只逼真料**：编的深度内行一眼看穿 |
| **审稿溯源** | 交稿前换独立视角，逐句标来源；指不出处的（串料/编造）打回 | **真实性机械核对**：每句事实/数字/对比都能追到素材；专业性只标疑点抛给你，AI 不下结论 |

> **为什么不替你编深度？** 专家恰恰一眼能看出编造的细节。AI 变不出它不懂的领域知识，
> 硬塞"看起来专业"的话只会更像 AI。Bragi 的办法是把空的地方亮出来、逼出你脑子里的真料——
> 有料了自然不肤浅，没有就老实标记，让你补。

## 一个真实例子

一段 AI 味很重的周报洞察（"全面赋能…系统性闭环…持续优化…"，破绽密度约 11/百字，
通篇没一个具体事实），Bragi 先问"给谁看"，再分别改给两个 leader：

- **给业务 leader**：第一句直接给判断（"行为护栏下季度值得推到内场试点"），
  配上真实的市场窗口，砍掉技术细节，试点成本不够就标缺料。
- **给研发 leader**：第一句给边界（"目前只做了运行时拦截，覆盖不到开发阶段"），
  点明一个内行才在意的区分，成本和误报率标缺料。

同一批事实，两个读者，两篇文章，且都没为显深度编一个数字。
完整对比见 [examples/before-after-weekly.md](skills/bragi/examples/before-after-weekly.md)。

## 安装

```bash
npx skills add uaandesign/bragi-ai-humanizer
```

装完，把草稿丢给 Claude Code 并说：

```text
用 bragi 把这段改得不那么 AI，这是给我们业务 leader 看的
```

其它会触发它的说法：

- 「这段太 AI 了，改得像人说的」
- 「降低一下 AI 感」
- 「这稿子给 leader 看，帮我按他关心的改」
- 「显得更专业、别这么泛」

## 你会得到什么

- **改写稿**（标明面向哪个角色）
- **改了什么 · 为什么**：去味铲了几处（附 lint 前后密度对比）、为这个角色提了什么砍了什么、
  几处泛化换成了真料
- **缺料清单**：哪几处只有你能补——这不是 Bragi 改不动，是那块真东西只在你脑子里

## 和"随手润色"的区别

| | 随手让 AI 润色 | Bragi |
|---|---|---|
| 改之前 | 直接就改 | **先问给谁看**，没受众不动笔 |
| 去 AI 味 | 常常越润越花、更假 | 减法：铲排比、空心大词、套话，可用 lint 量出降幅 |
| 面向读者 | 谁看都一样 | 按角色矩阵重排，同内容对不同人是不同文章 |
| 专业深度 | 编点术语显得深 | **不造料**：逼出你的真料，编不出的老实标缺料 |
| 产物 | 一版新文字 | 改写稿 + 改动理由 + 缺料清单 |
