# AI 腔破绽清单（AI Tells）

> 一眼能认出 AI 的痕迹，逐条铲。核心认知：**AI 味来自"加得太多"——
> 排比、形容词、过渡句、工整收尾。去味是减法，主要靠删和断，不是再堆辞藻。**
>
> **AI 味分两层，都要铲：**
> - **第一层 土味 slop**（下面一~五节）：赋能、闭环、排比、套话头尾——蠢萌的 AI 腔。
> - **第二层 聪明改写腔**（第六节）：你让 AI"说人话/写得有劲"时，它吐出来的那套
>   对仗金句、生造比喻、"一句话："开场——**它假装自己很像人，更难抓、更危险。**
>   铲完第一层别得意，重点查第二层（这是真实翻车点，见 `examples/`）。

## 一、结构层（最致命，先铲）

| 破绽 | 长什么样 | 怎么改 |
|---|---|---|
| **套话开头** | "随着…的快速发展"、"在数字化浪潮下"、"近年来" | 删掉，第一句直接给最重要的判断/事实 |
| **总分总骨架** | 开头预告"本文将从三个方面"、结尾"综上所述/总而言之" | 砍头尾的元叙述，让内容自己开始、自己结束 |
| **三段式强迫症** | 什么都凑成三点、三个排比、三个维度 | 有几点说几点；一个有力的点胜过三个凑数的 |
| **列点强迫症** | 连贯的判断被打碎成 bullet，每条还对仗 | 该用段落讲的用段落，bullet 只留给真·并列项 |

## 二、句式层

| 破绽 | 长什么样 | 怎么改 |
|---|---|---|
| **排比堆叠** | "不仅…更…"、"既…又…"、"从…到…，从…到…" | 拆成长短不一的句子，留最有力的一句 |
| **工整对仗收尾** | "既是起点，也是终点"、"是挑战，更是机遇" | 删掉这种平衡句，敢下单边判断 |
| **万能过渡句** | "值得注意的是"、"总的来说"、"不难发现"、"由此可见" | 直接删，让上下句自然相接 |
| **把话说满** | "全方位"、"系统性"、"端到端"、"一体化"地覆盖一切 | 承认边界，说清"做了什么、没做什么" |
| **中庸两头堵** | "既要…也要…"却不给取舍 | 表明立场，给出取舍和代价 |

## 三、词汇层（空心大词，命中即换或删）

> 这些词的共同点：**信息量约等于零，但显得很专业**。它们正是 AI 味的浓度来源。

赋能、抓手、组合拳、闭环、生态、范式、链路、心智、势能、沉淀、对齐、拉通、
颗粒度、方法论、底层逻辑、顶层设计、护城河、第一性原理、价值、维度、纵深、
全方位、系统性、深层次、强有力、显著、卓越、全面、深入、有力、扎实地、切实地

**改法**：能换成具体动作就换（"赋能业务" → "让客服少接 30% 重复来电"），换不出来就删。
一句话删掉所有形容词和这些大词后还剩什么，那才是它真正说了的东西。

## 四、语气层

| 破绽 | 长什么样 | 怎么改 |
|---|---|---|
| **形容词通胀** | 强大的、全面的、深入的、卓越的、显著的、重要的 | 删评价，用事实代替（"重要" → 给出它影响了什么） |
| **决心式空话** | "我们将持续优化"、"不断提升"、"致力于打造" | 删，或换成一个具体的下一步动作 |
| **过度礼貌/铺垫** | "首先非常感谢"、"接下来让我们一起" | 删客套，直接进入内容 |
| **解释显而易见** | 把人人都懂的概念再定义一遍 | 删，默认读者是懂行的人 |
| **emoji 与加粗通胀** | 每段配 emoji、关键词全加粗 | 留一两个真正的重点加粗，其余去掉 |

## 五、★ 聪明改写腔（第二层，铲完土味 slop 后重点查这一节）

这是 AI 被要求"说人话 / 写得有劲 / 更精炼"时吐出来的东西。它**没有空心大词，反而读着挺利落**——
正因如此最容易漏网。判别核心：**它在用形式（对仗、比喻、短句）制造"很有劲"的错觉，
但删掉这些形式后，没有新增任何具体事实。**

| 破绽 | 长什么样 | 怎么改 |
|---|---|---|
| **"一句话："开场** | "一句话：本项目把 X 变成 Y" / "说白了就是…" | 删掉这个壳，直接说那件事；真要一句话，就给一个带数字的事实 |
| **"把 A 从 X 变成 Y"公式** | "把质量从靠人自觉变成卡在流程里" | 换成具体做了什么 + 结果数字（"不交两张表不准写代码；对账 4h→1h"） |
| **生造比喻 / 起名癖** | "三道设计硬门"、"塌方"、"护城河"、"飞轮"、给步骤起个酷名字 | 直接叫它本来的名字（"三条强制检查"），别包装成隐喻 |
| **对仗反转金句** | "不是 X，而是 Y"、"既是起点，也是终点"、"从 A 到 B 的跃迁" | 拆成平实陈述，敢只说一边，别为对仗凑后半句 |
| **短句装狠 / 破折号抖包袱** | 一连串三五字短句制造节奏；"——而这，正是关键" | 该长就长，按意思断句，不为戏剧感切碎 |
| **金句密度过高** | 每段都想留一句可截图的话 | 一篇有一两个真观点就够，其余老实叙述；通篇金句 = AI |
| **空心对比/排比升级版** | "不止于…更在于…"、"从 0 到 1，从 1 到 N" | 给出 0 是什么、1 是什么、N 是什么的真实内容，否则删 |

> **辨别口诀**：把一句话里所有的**对仗、比喻、短句节奏**都拆平，看还剩什么。
> 剩下一个带数字/带场景的事实 → 留着；剩下一个空的判断 → 它本来就是 AI 在凹造型，删了补真料。
> 详细的"AI 腔 vs 人话"对照见 `references/voice.md`。

## 六、人话的样子（往这里改）

去味不只是删，还要往"像人说的"靠。人写的东西通常：

- **节奏不均**：长短句混着来，不是每句都一样长一样工整。
- **敢下判断**：直接说"我觉得这条路不对"，不和稀泥。
- **敢省略**：相信读者能接上，不把每个逻辑链都补全。
- **有具体**：有数字、有名字、有场景，不停在抽象层。
- **有视角**：有"我"的取舍和立场，不是中立的播报。
- **留白**：话不说满，留一点让读者自己想。

> 一个快速自测：把改写稿读出声。**像不像你会当面跟同事说的话？** 像，就对了；
> 还是像念稿、像公文、像颁奖词，就还有 AI 味没铲干净。

## 七、补充破绽（取自 Humanizer 的"AI 写作特征"清单）

> 这几条是上面没单独列、但很高频的，借鉴自 [blader/humanizer](https://github.com/blader/humanizer)
> 与 [op7418/Humanizer-zh](https://github.com/op7418/Humanizer-zh)（源自维基百科 AI Cleanup 项目）。
> 只补养分，判别标准不变：**删掉这层修饰后，有没有少掉一个具体事实？没有，就是 AI 在凹。**

| 破绽 | 长什么样 | 怎么改 |
|---|---|---|
| **系动词回避** | 用"作为 / 代表 / 标志着 / 充当 / 设有 / 拥有"绕开一个"是" | 直接用"是 / 有"。「X 作为团队的核心枢纽」→「X 是团队的枢纽」 |
| **"-ing / 着"尾巴假深度** | 句末挂一截分词补语："…，**反映了**更深的联系 / **体现着**…" | 砍掉这截尾巴；要么给出它具体反映了什么事实 |
| **模糊归因** | "专家认为 / 行业报告显示 / 观察者指出 / 业界普遍" | 给具体来源（谁、哪年、哪份）；给不出就删这句判断 |
| **同义词循环** | 同一个东西换着叫："主人公…主角…核心人物…该英雄…" | 统一叫一个名字，别为"不重复"硬换词 |
| **chatbot 残留** | "希望这对你有帮助""当然！""你说得对""如需我可以…" | 全删，这是对话壳，不是内容 |
| **过度限定堆叠** | "可能或许在某种程度上被认为大概会有一些影响" | 留一个限定词或都删，敢下判断 |
| **填充短语** | "为了实现这一目标 / 由于…的事实 / 在这个时间点 / 值得一提的是" | 换短的：「为此 / 因为 / 现在 / （直接删）」 |

> 注：humanizer 那套是通用文风清洗；**Bragi 的根基不是它，是"垂类纵深 + 对准角色"**——
> 去味只是地基那一层，真正决定稿子好不好的是后面两步。别把 Bragi 用成一个纯 humanizer。
